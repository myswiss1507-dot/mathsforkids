# Maths For Kids

An educational iOS app designed to help children aged 2-10 learn mathematics through interactive games.

## Features

- **Three Difficulty Levels:**
  - **Toddler (2-4 years):** Counting and number recognition
  - **Early School (5-7 years):** Addition and subtraction
  - **Older Kids (8-10 years):** Multiplication and division

- **Multi-language Support:** English, Spanish, French, German, Chinese, Portuguese, Hindi, Tamil, Telugu, Marathi, Bengali

- **Performance Tracking:** Parent Report with detailed session history, question-by-question breakdown, and performance analytics

- **Speed Bonus System:** Rewards for fast and accurate answers

- **Offline Functionality:** Works completely offline, no internet required

## Technical Details

- **Platform:** iOS 15.0+
- **Framework:** SwiftUI
- **Storage:** Local (UserDefaults)
- **Privacy:** No data collection, all data stored locally
